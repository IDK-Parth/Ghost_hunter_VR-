# Ghost Hunter VR 

A VR shooting game built with Unity.

## Features
- VR shooting mechanics
- Ghost enemy AI
- Ray-based weapons
- Oculus / XR support

## Tech Stack
- Unity
- C#
- XR Interaction Toolkit
- meta SDK for the development

## How to Run
1. Open project in Unity Hub
2. Open SampleScene
3. Build for meta Quest
